# 更纱黑体

更纱黑体的英文名为 Sarasa Gothic，是一套面向多语言程序员和技术写作场景的字体家族。它常被用于代码编辑器、终端和技术文档，尤其适合需要中文、日文、韩文与西文代码共同出现的环境。

## 定位

更纱黑体的核心价值是技术场景中的多语言等宽体验。普通英文字体在遇到中文 fallback 时，常出现中文偏小、行高不稳或宽度不协调的问题。更纱黑体通过面向 CJK 与拉丁代码字符的组合设计，降低这些混排问题。

## 风格特征

- 面向代码和终端环境，重视字符区分度。
- 提供多种变体，覆盖等宽、准等宽、UI 和终端等使用方式。
- 中英文混排相对协调，适合技术文档和代码注释。
- 字形气质中性，不抢内容本身的注意力。
- 文件和版本选择较多，需要按场景挑选具体变体。

## 适用场景

- IDE、编辑器、终端和日志查看器。
- 中文技术文档、代码示例和注释。
- 需要中英文宽度关系稳定的命令行界面。
- 程序员个人字体配置和团队开发环境推荐。

## 不适合场景

- 品牌标题、海报主视觉和强个性展示。
- 需要精致西文排版的长篇英文阅读。
- 大面积营销页面正文。
- 不需要 CJK 混排的纯英文代码环境中，可能不是最轻量选择。

## 排版建议

用于编辑器时，先确认选择的是 Mono、Term、Gothic 还是 UI 变体。终端更适合选择面向终端的版本，代码编辑器则可根据个人偏好选择是否启用连字、是否需要 Nerd Font 补丁版本。

中文技术文档可以使用更纱黑体展示代码块，正文仍可使用更适合阅读的中文黑体。这样能把代码环境和叙述内容区分开。

## 授权注意

更纱黑体项目以开源方式发布。由于它涉及多来源字体组件和多个变体，发布、嵌入或二次分发时应阅读仓库许可证文件，并保留原项目说明。
