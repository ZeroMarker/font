# 阿里巴巴普惠体

阿里巴巴普惠体是阿里巴巴推出的免费商用字体家族，定位为可用于多场景的通用正文字体。它的设计目标偏向现代、清晰、中性，适合 UI、品牌物料、正文排版和商业传播。

## 定位

阿里巴巴普惠体适合作为中文产品的主字体或品牌基础字体。相比展示字体，它的个性更克制，能承担较长时间阅读和复杂信息展示。对于需要统一线上线下物料的团队，它比临时混用系统字体更容易形成一致的视觉基调。

## 风格特征

- 现代黑体风格，笔画简洁，字面稳定。
- 字重体系较完整，方便建立标题、正文和强调层级。
- 适合中文正文，也能覆盖常见中西文混排。
- 气质中性，商业场景适应面较广。
- 相比系统字体，更容易在品牌物料中保持统一。

## 适用场景

- 产品 UI、官网、活动页和后台系统。
- 品牌物料、PPT、宣传册和电商页面。
- 中短正文、说明文案、标签和按钮。
- 需要免费商用基础字体的商业项目。

## 不适合场景

- 需要强烈手写感、复古感或实验风格的视觉项目。
- 需要专业代码阅读体验的 IDE 或终端。
- 需要高度个性化品牌识别的 logo 字体。
- 对特定少数文字系统覆盖有硬性要求但未验证字符集的场景。

## 排版建议

阿里巴巴普惠体可以作为中文字体栈中的主字体。UI 中建议明确区分 Regular、Medium、Bold 等字重，不要只依赖字号表达层级。正文行高可从 1.5 左右开始测试，标题可根据字号适度降低行高。

与英文品牌词、数字混排时，应检查数字高度、标点宽度和英文缩写的视觉重量。对于大量数据表格，建议测试是否需要搭配等宽数字或专用表格字体。

## 授权注意

阿里巴巴普惠体官方页面说明其可免费商用。正式项目仍应保存授权说明，并核对是否允许嵌入 App、Web、硬件设备、PDF 或其他具体载体。字体版本较多，团队内部应固定版本，避免不同成员输出文件不一致。
