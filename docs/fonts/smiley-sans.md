# 得意黑

得意黑的英文名常见为 Smiley Sans，是一款面向标题和展示用途的中文黑体。它的字形在人文手写感与几何黑体之间取平衡，整体字身偏窄，并带有明显的斜向动势，因此在大字号环境中具有较高识别度。

## 定位

得意黑更适合作为展示字体，而不是通用正文字体。它适合承担视觉锚点，例如海报标题、活动主视觉、封面文字、品牌辅助标题和社交媒体配图。字体本身有明确性格，能快速建立轻松、年轻、活跃的视觉氛围。

## 风格特征

- 字身偏窄，版面占宽相对克制，适合标题排版。
- 笔画与转折带有手绘美术字气质，不是完全中性的黑体。
- 字形有轻微斜向趋势，画面动感强。
- 大字号下个性明显，小字号下细节可能变得拥挤。
- 适合短句，不适合承载大段连续阅读。

## 适用场景

- 活动海报、营销 banner 和社交媒体封面。
- 品牌辅助视觉中的短标题。
- 年轻化、娱乐化、创意类产品的强调文字。
- 需要和中性正文字体形成反差的标题层级。

## 不适合场景

- 长正文、说明文档、合同、后台管理系统。
- 对严肃性、稳定性和中性表达要求较高的场景。
- 信息密度很高的小字号 UI。
- 大量数字、表格或代码内容。

## 排版建议

得意黑适合搭配中性的正文黑体，例如阿里巴巴普惠体、Noto Sans CJK 或系统默认黑体。标题使用得意黑时，正文应尽量降低个性，避免多个强风格字体互相竞争。

建议控制字距和行距：短标题可以略微收紧字距，但长标题应保留足够间距，避免斜向结构造成压迫感。用于移动端时，应重点检查 16px 以下字号的笔画粘连和标点表现。

## 授权注意

得意黑项目页显示采用开源字体授权。若用于商用项目、字体文件再分发、字体改作或嵌入产品，应查看项目仓库中的许可证文件和 Reserved Font Name 说明。基于 OFL 类许可证改作字体时，通常需要特别注意保留名称限制。
