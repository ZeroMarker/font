# Maple Mono

Maple Mono 是一款开源等宽字体，面向 IDE、终端和技术内容。它以圆角、连字、控制台图标和中英文混排支持为主要特点，适合喜欢柔和代码视觉风格的开发者。

## 定位

Maple Mono 适合个人开发环境、终端配置和技术文档代码展示。相比一些更硬朗的等宽字体，它的圆角设计让代码界面更柔和；同时提供 Nerd Font 相关能力和细粒度自定义选项，适合需要调校字体体验的用户。

## 风格特征

- 等宽字体，适合代码和终端。
- 字形带圆角，视觉更柔和。
- 支持代码连字，适合现代编辑器。
- 提供包含 Nerd Font 图标的版本，方便终端状态栏和命令行工具。
- 强调中英文宽度协调，适合中文开发者环境。

## 适用场景

- IDE、VS Code、Neovim、终端和命令行工具。
- 中文代码注释较多的开发环境。
- 需要图标字体的终端主题和状态栏。
- 喜欢圆角与轻松视觉风格的代码界面。

## 不适合场景

- 严肃正式的长篇正文。
- 需要传统机械感或极高中性代码字体的团队规范。
- 大量印刷正文和营销标题。
- 对连字或图标补丁有严格禁用要求的代码审查场景。

## 排版建议

Maple Mono 版本较多，使用前应确认是否需要 CN、NF、NL、Unhinted 等变体。终端用户通常需要 Nerd Font 版本，编辑器用户则应根据是否喜欢连字选择普通版或无连字版本。

如果项目中有大量中文注释，建议同时检查全角标点、中文宽度、行高和光标定位。不同终端对字体 fallback 和图标宽度处理不同，配置后应测试提示符、状态栏、表格和日志输出。
